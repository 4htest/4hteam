# 4H-Team
---
## client
#### 사용법

> npm install -g babel webpack webpack-dev-server

git clone을 하고 명령창에서
> cd 4hteam/client

로 이동.
> npm install

을 명령창에 입력하여 설치 완료 후
> npm start

을 입력하고 정상적으로 compile 됐을 시에

> localhost:7777

로 접속.

---
## server
#### 사용법

git clone을 하고 명령창에서
> cd 4hteam/server

로 이동.
> mvn spring-boot:run

서버 실행후
웹 브라우저 주소창에 localhost:8080 입력
